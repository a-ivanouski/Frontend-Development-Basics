My framework gives the opportunity to create MVC architecture.

Framework "Model" Behavior

My framework can get variable in "behavior" function and render her on view, for that you should add attribute in tag "framework-model="value""

Code

"js"
createBehavior( "frameworkModelBehavior" ,  function(self){
  self.value = "This is my framework value!!!";
})
"HTML"
< label framework-model="value"></label>

Example

Framework "Model Object" Behavior

Also framework model understanding properties of Objects

Code

"js"
createBehavior( "frameworkModelObjectBehavior" , function(self){
  self.value = "This is my framework value!!!";
self.obj = {
  value: {
   value: "This is obj.value.value"
  }
 };
})
"HTML"
< label framework-model="obj.value.value"></label>
