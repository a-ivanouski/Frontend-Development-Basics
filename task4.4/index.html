My framework gives the opportunity to create MVC architecture.
All you need is implement "createBehavior" function.

Framework "Model" Behavior

My framework can get variable in "behavior" function and render her on view, for that you should add attribute in tag "framework-model="value""

Code

"js"
createBehavior( "frameworkModelBehavior" ,  function(self){
  self.value = "This is my framework value!!!";
})
"HTML"
< label framework-model="value"></label>

Example

Framework "Model Object" Behavior

Also framework model understanding properties of Objects

Code

"js"
createBehavior( "frameworkModelObjectBehavior" , function(self){
  self.value = "This is my framework value!!!";
self.obj = {
  value: {
   value: "This is obj.value.value"
  }
 };
})
"HTML"
< label framework-model="obj.value.value"></label>

Framework "Click" Behavior

Framework is understanding mouse click on element.

Code

"js"
createBehavior("frameworkClickBehavior", function(self){
 var tmp = false;
 self.click=function(){
  tmp=!tmp;
  self.valueClick= tmp ? "Click" : "Not click";
 }

 self.click();
})
"HTML"
<button class="click-button" framework-click="click()" framework-model="valueClick"></button>

Framework "Show" Behavior

Framework can hide or show wome element.

Code

"js"
createBehavior("frameworkShowBehavior", function(self){
 self.showBlock = false;
 self.show = function(){
  self.showBlock = !self.showBlock;
 }
})
"HTML"
 <div class="rel">
  <button class="click-button" framework-click="show()">Show Block</button>
  <div class="block" framework-show="showBlock"></div>
 </div>
"CSS"
.click-button{
 padding: 10px 20px 10px 20px;
 border: none;
 background-color: lightgreen;
}

.rel{
 position: relative;
}

.block{
 display: inline-block;
 position: absolute;
 left: 135px;
 top: 0px;
 height: 35px;
 width: 100px;
 background-color: yellow;
}
