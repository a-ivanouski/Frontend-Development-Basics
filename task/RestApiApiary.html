    RestApi

    Introduction

    RestApi is a Rest API allowing consumers to manage tasks. It consists of two main parts - category api and task api.

    This resource have one param which represents category id.

    Task Api

    Retrieve/delete the task

    This resource have one param which represents task id.

    A task object has the following attributes:

    • Id

    • Description

    • Deadline

    • Priority - integer param which represents task Priority(1 - Hi; 2 - Medium; 3 - Low)

    • Status - integer param which represents task Status(1 - Normal; 2 - Completed; 3 - Overdue)

    • Category - category object
