Task SASS Framework
This frame work use Eric Meyer’s “Reset CSS” 2.0 for css reset. This is most for base css reset. You can get it from here http://cssreset.com/.
The framewrok use http://fonts.googleapis.com/css?family=Lato font. It also provide next clases for font size: font-size-large-3, font-size-large-2, font-size-large-1, font-size-medium, font-size-normal, font-size-small.
Example:
<h1 class="font-size-large-3"></h1> <h2 class="font-size-large-2"></h2> <h3 class="font-size-large-1"></h3> <h4 class="font-size-medium"></h4> <h5 class="font-size-normal"></h5> <h6 class="font-size-small"></h6>

The framework provides base classes for form elements.
Example:
<input class="btn info" type="button" value="button" /> <textarea class="input"></textarea> <button class="btn ok"></button> <select class="select"> <option></option> <option>open</option> <option>closed</option> </select> <input type="radio" id="radio" name="radio1" class="radio" /> <label for="radio"><span></span></label> <input type="checkbox" name="checkbox" id="checkbox" /> <label for="checkbox"><span></span></label>
The framework has grid system based on full screen and provides classe wich allow to issert elements 1/12 to 1 of the screen. Also it supports 3 screen resolution ratio betwen them is 1/2 and 1/4 for mobile.
Example:
<div class="column-6"></div> <div class="column-5"></div> <div class="column-4"></div> <div class="column-3"></div> <div class="column-2"></div> <div class="column-1"></div>
The framework has classes wich helps hide elements for differen devices.
Example:
<div class="hide-tablet"></div> <div class="hide-phone"></div> <div class="hide-desk"></div>
