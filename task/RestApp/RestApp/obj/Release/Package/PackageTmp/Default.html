    RestApi

    Introduction

    RestApi is a Rest API allowing consumers to manage tasks. It consists of two main parts - category api and task api.

    Reference

    Category Api

    Retrieve/delete the category

    This resource have one param which represents category id. To create and update category object pass through request body.

    A Category object has the following attributes:

    Task Api

    Retrieve/delete the task

    This resource have one param which represents task id. To create and update task object pass through request body.

    A task object has the following attributes:

    • Id

    • Description

    • Deadline

    • Priority - integer param which represents task Priority(1 - Hi; 2 - Medium; 3 - Low)

    • Status - integer param which represents task Status(1 - Normal; 2 - Completed; 3 - Overdue)

    • Category - category object
